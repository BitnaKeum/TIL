
## 구조체

- 구조체가 존재하진 않지만, @dataclass 데코레이션과 class를 이용해 만들 수 있다 _(Python 3.7부터)_

```
from dataclasses import dataclass

@dataclass
class Person:
  name: str = None
  age: int = None
```
```
bitna = Person("bitna", 24)
```
